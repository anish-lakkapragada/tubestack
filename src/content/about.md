# tubestack feature guide 

tubestack is a minimal desktop application to enable you to read youtube content in a more clean, organized way. it is designed to cultivate self-reflection and intentful exploration in one's interests. 

below is a guide explaining a little bit on how tubestack works and some features we hope you enjoy. 

## how tubestack works 

tubestack asks youtube for the same public information your browser already sees, and then formats the captions into a prettified transcript. **everything stays local and free, with no cloud nor api keys required.** 

## read later & history 

history tracks what you've recently read for your own perusal. 

read later allows you to put things in a queue (first in, first out) for you to go through. you can toggle {{icon:bookmark}} on any article to add or remove it from your read later list. 

also, you can click `upcoming` to view the next thing on your list to read.

## locked reading mode 

sometimes we need to stop searching and focus on what's in front of us. 

enabling locked {{icon:lock-icon}} mode triggers full-screen mode and forces you to read the article you are currently on by disabling search and other features. run it for five, ten, or twenty minutes.
